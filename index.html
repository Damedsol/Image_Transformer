Opciones de accesibilidad

Aumenta el contraste de colores para mejor legibilidad
Cambia el tamaño del texto en toda la aplicación
Utiliza una fuente diseñada para facilitar la lectura a personas con dislexia
Minimiza o elimina las animaciones y efectos de movimiento
Ajusta el espaciado entre letras, palabras y líneas de texto

Atajos de teclado

  • Ctrl + Alt + A: Abrir/cerrar opciones de accesibilidad
  • Ctrl + Alt + C: Alternar modo de alto contraste
  • Ctrl + Alt + D: Cargar una imagen desde el dispositivo
  • Ctrl + Alt + S: Iniciar conversión de imagen

Nota: En Mac, usa Cmd en lugar de Ctrl
