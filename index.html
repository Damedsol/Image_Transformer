Accessibility options

Increases color contrast for better readability
Changes text size throughout the application
Uses a font designed to facilitate reading for people with dyslexia
Minimizes or eliminates animations and motion effects
Adjusts spacing between letters, words and text lines

Keyboard shortcuts

  • Ctrl + Alt + A: Open/close accessibility options
  • Ctrl + Alt + C: Toggle high contrast mode
  • Ctrl + Alt + D: Load an image from the device
  • Ctrl + Alt + S: Start image conversion

Note: On Mac, use Cmd instead of Ctrl
